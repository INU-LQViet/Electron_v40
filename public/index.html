About project.

Electromyography (EMG) is a technique for evaluating and recording the electrical activity produced by skeletal muscles. The EMG can be used to control robot or game instead of the traditional devices such as keyboards,... by collecting signal from person's muscle. In particularly, EMG is the promissing choice when physical movement is not possible, such as people have injured upper limbs. This project adopts a application which controls a video game by using EMG signals. Similar project can be reference [1], [2], [3]. The principle of application can be shown in figure 1:

The desktop application will plot the EMG signal and use this signal to control the character of game. The position and speed of character depend on the intensity of signal (FUZZY control is used).

Bird Game

The EMG signal is used to control the character (bird) in the bird game. The intensity of signal will effects the y-position and y-speed of the bird. After the bird pass a pipe, the scores will increase 1.

Conclusion

This research presented a potential method to control a video game by using directly the EMG signal from muscle. The application have used a simple FUZZY method to convert from EMG signal to character motion in video game. In the future work, multiple EMG sensors will be set up to observe more EMG signal from various muscles. Also, the machine learning algorithm will be set up to recogize hand's gestures. Moreover, the EMG can be used to control a manupulator robot like UR3.
